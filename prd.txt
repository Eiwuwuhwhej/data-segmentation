Backend: Python Flask API (hosted separately)

5. Functional Requirements
5.1 File Upload (Frontend)
Accept .csv files only
Validate file size (<10MB)
Show error if invalid
5.2 API Communication
Endpoint: /analyze
Method: POST
Payload: FormData (CSV file)
5.3 Data Processing (Backend)

Steps:

Read CSV using Pandas
Drop missing values
Convert dates
Aggregate per customer
5.4 RFM Calculation
Recency тЖТ Days since last purchase
Frequency тЖТ Number of transactions
Monetary тЖТ Total spend
5.5 Clustering
Standardize RFM values
Apply K-Means (k = 3)
Assign labels:
High Value
Medium Value
Low Value
